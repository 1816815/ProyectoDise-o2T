En este post vamos a hacer una demostración práctica de los filtros en CSS aplicados sobre una misma imagen. Como referencia, tomaremos esta imagen del logo de CSS con el significativo color Rebeccapurple.

Los filtros pueden influir mucho a la hora de definir la experiencia de usuario. Además, manipular una imagen de este modo es mucho más eficiente para un desarrollador web que tener que editar una misma imagen y subirla en distintos estados. En Bootstrap 5 no hay clases específicas para aplicar filtros a una imagen, pero como se ha visto, aplicar un filtro en css no ocupa más de una línea.

Como dato curioso, si haces click derecho en una de las imágenes de prueba y seleccionas "Abrir imagen en una pestaña nueva", se te mostrará la imagen sin filtros aplicados. Estarás accediendo directamente a la referencia del archivo, sin css aplicable.

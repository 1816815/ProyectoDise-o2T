Imágenes y CSS

Propiedades de fondo (background)

La propiedad background-image permite establecer una imagen como fondo de un elemento.

En este ejemplo, hemos usado background-size: cover para que la imagen cubra todo el contenedor, y background-position: center para centrarla.

Background Attachment

La propiedad background-attachment determina si la imagen de fondo se desplaza con el contenido o permanece fija.

Aquí, hemos usado background-attachment: fixed para que la imagen permanezca fija mientras se desplaza la página.

Objetos en CSS

Las propiedades object-fit y object-position permiten controlar cómo se ajusta una imagen dentro de su contenedor.

Object View Box

La propiedad object-view-box permite definir un área de visualización específica para la imagen, similar a la propiedad viewBox en SVG.
